With Some Simple Text:

Using #+BEGIN_SRC org makes the output be highlighted using whatever highlighting is active for org-mode, which may include some personal configuration. 

 1: #+TITLE: Holy Cow!
 2: 
 3: #+REVEAL_HLEVEL: 2
 4: #+REVEAL_THEME: solarized
 5: #+OPTIONS: toc:nil num:nil
 6: # Suppress the normal customized CSS I use with org:
 7: #+HTML_HEAD:
 8: 
 9: * Any Fool* Can Make a Presentation
10: 
11: #+ATTR_REVEAL: :frag t
